Stadia ViGEm — Portable (x64)
==============================

Map a Google Stadia controller (USB or Bluetooth) to a virtual Xbox 360 gamepad.

REQUIREMENTS (install once on this PC)
------------------------------------
- ViGEmBus: https://github.com/ViGEm/ViGEmBus/releases
- HidHide:  https://github.com/nefarius/HidHide/releases

Or use StadiaViGEm-Setup-x64.exe from the same GitHub release — it installs drivers automatically.

USAGE
-----
1. Run stadia-vigem-x64.exe (tray icon appears).
2. Connect your Stadia controller (USB or Bluetooth).
3. If you see double input, whitelist this exe in HidHide Configuration Client.

NOTES
-----
- USB: input + vibration
- Bluetooth: input only (Windows 11 OS limitation)
- Debug log: debug.log next to the executable

Full docs: https://github.com/lukasfuscic19/StadiaEmu
